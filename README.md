# NEU Course Registration Tool (Deprecated)
This project was developed as a personal tool to automate course registration on the old NEU registration system.

## Status
This project is no longer functional.
The NEU course registration website has been completely redesigned and updated, which makes this tool incompatible with the current system.

## Note
- The project has been officially discontinued.
- No further updates or maintenance will be provided.
- The code is kept here for reference and learning purposes only.
- In case you want to see how it worked, please visit: [link here](https://drive.google.com/file/d/17iEFaQ8cZ96rWGCtgGShl7n8GVEplX8p/view?usp=sharing)
